# plexplay
Playlist management - scan through Plex libraries and manage music playlists.

The core design assumption has on music libraries and playlists but it might 
be possible to handle other media (but don't plan on it).

# Installation
1. Install Python 3.7+
1. Install a virtual environment
1. Install packages from `requirements.txt`

# Usage
This is being built primarily as a command line application.
